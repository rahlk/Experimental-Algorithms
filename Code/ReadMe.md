### Run
To run a certain algorithm **algo** on certain **n** processors
```
sh runner.sh n algo
```
where **algo** in **["de", "gale"]** and **n** <= **16**

### Benchmark
To benchmark an **algo** between **2** and **n** processors
```
sh benchmark.sh n algo
```
where **algo** in **["de", "gale"]** and **n** <= **16**
